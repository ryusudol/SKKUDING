## How to start projects
### Clone this repository
```JS
git clone https://github.com/ryusudol/SKKUDING.git
```
### Move to a project branch
```JS
git checkout <branch>
```
### Install node packages
```JS
npm install
```
### Run the project
```JS
npm start
```
